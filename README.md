## Overview
This project was developed as part of a university term paper in Business Information Systems.
It focuses on the design and implementation of a relational database for an online ticketing platform.

## Key Features
- Relational database design (MySQL)
- Support for artists, tours, venues, events, and customers
- Flexible seating and standing area modeling
- Constraints and triggers to ensure data integrity
- Test data generation and example queries

## Technologies
- MySQL
- SQL

## How to Use
1. Import the SQL file into a MySQL environment
2. Execute the script to create the database and insert test data
3. Run the example queries to explore the data model
